This code is a matlab code, which has been tested on 32bit and 64bit Windows systems. If you find any bugs, please contact Hongkai Yu (hongkaiyu2012@gmail.com). This code is a simplified version for the following ICIP 2014 paper:

Hongkai Yu, Min Xian, and Xiaojun Qi, "Unsupervised Co-Segmentation Based on a New Global GMM Constraint in MRF", IEEE International Conference on Image Processing (ICIP), 2014.

If you use this code or are inspired by our work, please cite our paper. 

++++++++++++++++++++++++++++++++++++++++++++++++++++++
Following are external packages used in our code.  
[1]. Co-saliency: 
Huazhu Fu, Xiaochun Cao, Zhuowen Tu, "Cluster-based Co-saliency Detection",  IEEE Transactions on Image Processing, 2013.

[2]. Maxflow algorithm for optimization: 
Y. Boykov and M.-P. Jolly, “Interactive graph cuts for optimal boundary region segmentation of objects in nd
images,” IEEE International Conference on Computer Vision (ICCV), 2001, pp. 105–112.
++++++++++++++++++++++++++++++++++++++++++++++++++++++
Running the code:
put input images in the 'Images' folder. 
put the co-saliency detection results by [1] in the 'CoSaliency' folder. 
run 'Coseg_main.m'.
